## Chapter 1 - Reliable, Scalable, and Maintainable Applications

### Thinking About Data Systems

New tools for data storage and processing have emerged. They are optimized for a variety of different use cases, and they are combined to form a *data system* and *to provide a service*. The data system should be reliable, scalable, and maintainable.
